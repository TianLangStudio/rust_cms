# 运行 
**请先安装Rust**  

> git clone git@github.com:TianLangStudio/rust_cms.git     
> cd rust_cms 
- 创建Mysql数据库　导入doc/rust_cms.sql  
- 修改conf/application_dev.yaml文件中的数据库链接配置　
> cargo run
